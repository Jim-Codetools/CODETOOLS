3. Compliance with NZ Building Code:

This calculator is designed to assist the user in verifying compliance with NZ building code clause H1 (Specifically Objective H1.1; Functional Requirement H1.2a; and Performance Criteria H1.3.1a-b, H1.3.2e, and H1.3.3a-f). This calculator is based on the methodology set out in H1/AS1 - 5th Edition, Amendment 1.



4. Usage Limitations:

This calculator can be used to analyze housing and other buildings with an area of occupied space no greater than 300 m2. It cannot be used to analyze buildings that include a suspended floor as part of the thermal envelope. It cannot be used for buildings where an appropriate user option is not available. Note that users are allowed to select a more conservative option if an exact one is not available.



5. Construction R Values:

Construction R Values are calculated in accordance with NZS 4214:2006. To enhance the user experience, this calculator makes certain assumptions about the composition of building elements and resulting construction R Values. Users are responsible for understanding these assumptions and their significance in assessing the calculator's suitability. These assumptions are conservative and generally minor in nature. Consequently, a building may perform marginally better than these results indicate. Refer to the calculator's technical justification for a detailed description of these assumptions.
